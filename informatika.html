Elemen 1 & 2

Berpikir Komputasional & TIK

Fondasi logika pemecahan masalah dan keterampilan produktivitas digital menggunakan aplikasi perkantoran modern.

BK — Pencarian & Pengurutan

Searching & Sorting

  • Linear Search: Memeriksa elemen satu per satu dari awal hingga akhir. Kompleksitas O(n). Cocok untuk data kecil atau tidak terurut.
  • Binary Search: Membutuhkan data terurut; membagi dua rentang pencarian tiap langkah. Kompleksitas O(log n). Jauh lebih cepat untuk data besar.
  • Bubble Sort: Membandingkan pasangan elemen bersebelahan dan menukarnya jika salah urutan. O(n²) — sederhana tapi lambat untuk data besar.
  • Selection Sort: Mencari elemen terkecil lalu menempatkannya di posisi paling depan, diulang hingga semua terurut. O(n²).
  • Insertion Sort: Menyisipkan elemen ke posisi yang tepat dalam bagian yang sudah terurut. Efisien untuk data hampir terurut.
BK — Struktur Data Linier

Stack & Queue

  • Stack (Tumpukan): Prinsip LIFO (Last In, First Out). Elemen terakhir masuk adalah yang pertama keluar. Operasi utama: push (masuk) dan pop (keluar). Contoh nyata: tombol Undo pada aplikasi, riwayat halaman browser.
  • Queue (Antrean): Prinsip FIFO (First In, First Out). Elemen pertama masuk adalah yang pertama keluar. Operasi: enqueue (masuk) dan dequeue (keluar). Contoh: antrean cetak dokumen (print queue), sistem tiket online.
  • Deque (Double-ended Queue): Kombinasi stack dan queue; elemen bisa masuk/keluar dari kedua ujung.
# Stack sederhana di Python
stack = []
stack.append("A")   # push
stack.append("B")
stack.pop()          # keluar: "B" (LIFO)

# Queue dengan deque
from collections import deque
q = deque()
q.append("X")        # enqueue
q.popleft()         # keluar: "X" (FIFO)
TIK — Aplikasi Perkantoran

Integrasi & Fitur Lanjut

  • Pengolah Kata (Word/Docs): Fitur lanjut meliputi mail merge untuk surat massal, styles & heading untuk dokumen terstruktur, daftar isi otomatis (TOC), komentar & lacak perubahan untuk kolaborasi.
  • Pengolah Angka (Excel/Sheets): Fungsi lanjut: VLOOKUP/XLOOKUP, IF bersarang, pivot table untuk analisis data, conditional formatting, pembuatan grafik dinamis.
  • Presentasi (PowerPoint/Slides): Slide master untuk konsistensi desain, animasi transisi, hyperlink antar slide, embed video/audio, dan kolaborasi real-time.
  • Manajemen Dokumen: Penggunaan folder dan nama file yang sistematis, version control sederhana, backup ke cloud (Google Drive, OneDrive).
Elemen 3 & 4

Sistem Komputer & Jaringan Internet

Memahami cara kerja komputer dari dalam, mulai dari komponen perangkat keras hingga bagaimana data berpindah melalui jaringan lokal dan internet global.

SK — Perangkat & Sistem

Komponen Sistem Komputer

  • CPU (Central Processing Unit): Otak komputer. Terdiri dari ALU (Arithmetic Logic Unit) untuk kalkulasi, CU (Control Unit) untuk koordinasi, dan register sebagai memori sementara ultra-cepat.
  • Memori: RAM (Random Access Memory) — volatile, digunakan saat program berjalan. ROM — non-volatile, menyimpan firmware/BIOS. Cache (L1/L2/L3) — lebih cepat dari RAM, lebih dekat ke CPU.
  • Penyimpanan: HDD (Hard Disk Drive) — magnetik, kapasitas besar, lambat. SSD (Solid State Drive) — flash memory, sangat cepat, lebih tahan banting.
  • Sistem Operasi: Program yang mengelola sumber daya hardware dan menjadi perantara antara pengguna dengan hardware. Contoh: Windows, Linux, macOS, Android.
  • Perangkat I/O: Input (keyboard, mouse, scanner, mikrofon) dan Output (monitor, printer, speaker).
JKI — Jaringan Dasar

Konsep Jaringan Komputer

  • Jenis Jaringan: LAN (Local Area Network) — area kecil (rumah/sekolah). WAN (Wide Area Network) — area luas (antarkota/negara). MAN (Metropolitan Area Network) — area kota.
  • Topologi Jaringan: Star (semua terhubung ke switch/hub pusat — paling umum), Bus (satu kabel utama), Ring (melingkar), Mesh (saling terhubung penuh).
  • Protokol: TCP/IP adalah protokol utama internet. HTTP/HTTPS untuk web. FTP untuk transfer file. SMTP/POP3/IMAP untuk email.
  • IP Address: Alamat unik setiap perangkat di jaringan. IPv4 (32-bit, contoh: 192.168.1.1). IPv6 (128-bit, untuk mengatasi keterbatasan IPv4).
JKI — Keamanan Jaringan

Komunikasi Data & Keamanan

  • Enkripsi: Mengubah data asli (plaintext) menjadi data tersembunyi (ciphertext) agar tidak bisa dibaca pihak tak berwenang. HTTPS menggunakan enkripsi TLS/SSL.
  • Firewall: Sistem yang memfilter lalu lintas jaringan berdasarkan aturan tertentu untuk mencegah akses tidak sah.
  • Ancaman Siber: Malware (virus, worm, trojan), Phishing (penipuan via email palsu), Ransomware (mengenkripsi data dan meminta tebusan), DDoS (membanjiri server dengan permintaan palsu).
  • Praktik Aman: Gunakan password kuat + 2FA, perbarui software secara rutin, hindari WiFi publik untuk transaksi penting, backup data berkala.
Elemen 5 & 6

Analisis Data & Algoritma Pemrograman

Dari pengolahan data mentah menjadi informasi bermakna, hingga menulis kode program prosedural yang menyelesaikan masalah nyata.

AD — Pengolahan Data

Analisis & Visualisasi Data

  • Siklus Data: Pengumpulan → Pembersihan (cleaning) → Transformasi → Analisis → Visualisasi → Interpretasi.
  • Statistik Deskriptif: Mean (rata-rata), Median (nilai tengah), Modus (nilai paling sering muncul), Range (selisih max-min), Standar Deviasi (sebaran data).
  • Visualisasi: Grafik batang (perbandingan kategori), grafik garis (tren waktu), pie chart (proporsi), scatter plot (korelasi dua variabel), histogram (distribusi frekuensi).
  • Spreadsheet untuk Analisis: Fungsi AVERAGE, MEDIAN, MODE, STDEV, COUNTIF, SUMIF. Pivot table untuk agregasi data multidimensi.
  • Interpretasi: Menarik kesimpulan yang didukung data; membedakan korelasi dan kausalitas.
AP — Algoritma

Dasar-Dasar Pemrograman

  • Variabel & Tipe Data: Integer, Float, String, Boolean. Penamaan variabel harus deskriptif (snake_case di Python, camelCase di C).
  • Percabangan: if, elif, else — mengeksekusi blok kode berdasarkan kondisi true/false.
  • Perulangan: for (jumlah iterasi diketahui), while (berulang selama kondisi terpenuhi). Hindari infinite loop.
  • Fungsi/Prosedur: Blok kode yang dapat dipanggil berulang dengan nama. Meningkatkan modularitas dan keterbacaan kode.
  • Array/List: Menyimpan banyak nilai dalam satu variabel, diakses via indeks (dimulai dari 0).
# Contoh program Python sederhana
def cek_nilai(nilai):
    if nilai >= 90:
        return "A"
    elif nilai >= 75:
        return "B"
    else:
        return "C"

data = [85, 92, 70, 88, 95]
for n in data:
    print(n, "->", cek_nilai(n))
AP — Bahasa C

Pemrograman Prosedural C

  • Struktur Program C: Setiap program diawali dengan #include untuk library, fungsi main() sebagai titik masuk, dan setiap pernyataan diakhiri titik koma.
  • Tipe Data C: int (bilangan bulat), float/double (desimal), char (karakter tunggal), void (tidak mengembalikan nilai).
  • Input/Output: printf() untuk menampilkan output, scanf() untuk membaca input pengguna. Format specifier: %d (int), %f (float), %c (char), %s (string).
  • Pointer: Variabel yang menyimpan alamat memori variabel lain. Konsep dasar untuk memahami alokasi memori dinamis.
// Program C sederhana
#include <stdio.h>

int jumlah(int a, int b) {
    return a + b;
}

int main() {
    int x = 5, y = 3;
    printf("%d\n", jumlah(x, y));
    return 0;
}
Elemen 7 & 8

Dampak Sosial & Praktik Lintas Bidang

Memahami tanggung jawab etis pengguna teknologi dan menerapkan seluruh pengetahuan informatika dalam proyek kolaboratif yang memecahkan masalah nyata.

DSI — Etika & Hukum

Etika Penggunaan Teknologi

  • Hak Kekayaan Intelektual: Hak cipta (copyright) melindungi karya digital. Menggunakan karya orang lain tanpa izin adalah pelanggaran. Pahami lisensi Creative Commons dan Open Source.
  • Privasi Data: Data pribadi (nama, alamat, nomor HP, foto) harus dijaga kerahasiaannya. UU ITE di Indonesia mengatur perlindungan data digital.
  • Hukum Siber (Cyber Law): UU No. 11 Tahun 2008 tentang ITE dan perubahannya mengatur kejahatan digital: penyebaran hoaks, konten negatif, akses ilegal, dan penipuan online.
  • Netiquette: Etika berinternet — tidak menyebar hoaks, tidak cyberbullying, menghargai privasi orang lain, menggunakan bahasa santun di ruang digital.
  • Sejarah Komputer: Generasi 1 (tabung vakum, 1940-an) → Gen 2 (transistor) → Gen 3 (IC/chip) → Gen 4 (microprocessor) → Gen 5 (AI & komputasi awan).
DSI — Keamanan & Digital

Keamanan Data & Literasi Digital

  • Identitas Digital: Jejak digital bersifat permanen — setiap unggahan, komentar, dan aktivitas online dapat tersimpan dan berdampak jangka panjang pada reputasi.
  • Hoaks & Disinformasi: Cara verifikasi: cek sumber terpercaya (BNPB, Kemkominfo, media nasional), gunakan reverse image search, perhatikan tanggal dan konteks berita.
  • Social Engineering: Teknik manipulasi psikologis untuk mencuri informasi sensitif. Contoh: phishing email, smishing (via SMS), vishing (via telepon).
  • Kesehatan Digital: Kelola waktu layar, istirahat dari perangkat (digital detox), jaga postur saat menggunakan komputer, waspadai dampak psikologis media sosial.
PLB — Proyek Kolaboratif

Praktik Lintas Bidang

  • Tujuan PLB: Mengintegrasikan semua elemen informatika untuk menyelesaikan permasalahan nyata yang relevan dengan kehidupan siswa dan masyarakat.
  • Tahapan Proyek: (1) Identifikasi masalah, (2) Perencanaan solusi, (3) Pembagian tugas tim, (4) Implementasi menggunakan tools informatika, (5) Uji coba & evaluasi, (6) Presentasi hasil.
  • Contoh Proyek: Membuat aplikasi sederhana pencatat nilai (Python), membuat website informasi sekolah (HTML/CSS), menganalisis data survei menggunakan spreadsheet, atau merancang sistem jaringan sekolah sederhana.
  • Kolaborasi & Dokumentasi: Gunakan platform seperti Google Workspace, GitHub, atau Notion untuk kerja tim. Dokumentasi proyek sangat penting untuk evaluasi.
  • Computational Thinking dalam PLB: Decomposition (pecah masalah), Pattern Recognition (kenali pola), Abstraction (fokus inti), Algorithm Design (rancang solusi).
